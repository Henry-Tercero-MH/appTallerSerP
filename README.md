# GestorERP — Mangueras del Sur

Sistema ERP de escritorio offline para talleres y negocios pequeños.  
**Plataforma:** Electron 30 + React 18 + SQLite · Windows / Linux

---

## Inicio rápido

```bash
cd frontend
npm install
npm run dev
```

> Si aparece `cabecera ELF inválida`: `npx electron-rebuild -f -w better-sqlite3`
